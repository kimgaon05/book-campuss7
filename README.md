# 📚 북캠퍼스 (BookCampus)
> **대학생 전용 중고 전공서적 거래 및 대여 플랫폼** > 코딩 에이전트와의 협업을 위한 제품 요구사항 정의(PRD) 및 문제 분해 기반 핵심 기능 구현 프로토타입입니다.

## 1. 프로젝트 개요 (Overview)
- **배경:** 매 학기 비싼 전공서적 구입비에 부담을 느끼는 대학생들을 위해, 같은 학교 학생 간 안전하고 빠르게 중고 서적을 거래하거나 대여할 수 있는 웹 서비스입니다.
- **목표:** 사용자가 원하는 책을 최소한의 단계로 찾고, 사기 우려 없이 안전하게 직거래 및 비대면 거래를 마칠 수 있도록 돕습니다.

## 2. 핵심 문제 분해 (Feature Decomposition)
수업 시간에 학습한 **문제 분해(Problem Decomposition)** 방식을 적용하여 핵심 기능을 컴포넌트와 상태 단위로 쪼개어 구현했습니다.
1. **도서 검색 및 상세 조회:** 검색어 입력 ➡️ 필터링 ➡️ 상세 정보 노출
2. **중고 판매/대여 글 작성:** 도서 정보 입력 ➡️ 상품 상태/조건 설정 ➡️ 유효성 검사 및 등록
3. **실시간 채팅 및 거래 예약:** 채팅방 개설 ➡️ 메시지 송수신 ➡️ 거래 약속 및 상품 상태 전환 (`[판매중]` ➡️ `[예약중]` ➡️ `[거래완료]`)

## 3. 기술 스택 (Tech Stack)
- **Frontend:** React.js (JavaScript, CSS)
- **State Management:** React Hooks (`useState`, `useEffect`)
